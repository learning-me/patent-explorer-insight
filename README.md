# Patent Search & Analytics Hu

The Patent Search & Analytics Hub is a powerful, user-friendly application designed to streamline the process of patent searching and analysis. It provides a comprehensive platform for inventors, researchers, and legal professionals to explore patent data, gain insights, and stay ahead of the curve.

## Features

- **Advanced Patent Search**: Quickly search for patents using keywords, patent IDs, or other criteria.
- **Detailed Patent View**: Access detailed information for each patent, including abstracts, assignee details, and technology classifications.
- **Interactive Analytics**: Visualize patent data with interactive charts and graphs to identify trends and patterns.
- **Responsive Design**: Enjoy a seamless experience across all your devices, from desktops to smartphones.

## Getting Started

To run this project locally, follow these steps:

1. **Install Dependencies**
   ```sh
   npm install
   ```
2. **Start the Development Server**
   ```sh
   npm run dev
   ```
